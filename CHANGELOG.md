# Changelog

All notable changes to this starter are tracked here.

This project has not published a tagged release yet. Until the first release, the `main` branch is the source of truth.

## Unreleased

### Added

- Public starter README with English default and Korean companion README.
- MIT license source file.
- Agent behavior guardrails adapted for product-neutral starter use.
- Agent workflow operating model for task briefs, planning, tool hygiene, parallel work, and verification.
- README migration checklist for turning the starter landing page into a project-specific landing page.
- Clone cleanup checklist for removing starter-specific history after reuse.
- Baseline validation through `./init.sh`.
- Public contribution, security, changelog, and starter copy simulation documents.
- Google `design.md` compatible `DESIGN.md` baseline and project-local `design-md` Codex skill.
- Design-agent documentation patterns for source-of-truth order, scoped `DESIGN.md` files, source evidence, and `Adopt` / `Do Not Adopt` reference models.
- Cross-agent skill bridge that exposes committed `.codex/skills` to Claude Code through `.claude/skills`.
- Product-neutral technical decision summary, decision backlog, validation ladder, and visual validation templates.
