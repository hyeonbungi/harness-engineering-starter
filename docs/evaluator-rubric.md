# Evaluator Rubric

완료 선언 전에 이 루브릭으로 결과를 평가합니다. 자기 확신이 아니라 증거를 기준으로 판단합니다.

| 범주 | 질문 | 점수 0-2 | 비고 |
| --- | --- | --- | --- |
| 정확성 | 요청된 동작 또는 문서 산출물이 실제로 충족됐는가? |  |  |
| 검증 | 필요한 검사가 실행됐고 증거가 남았는가? |  |  |
| 범위 규율 | 선택한 기능 또는 계획 범위 안에 머물렀는가? |  |  |
| 신뢰성 | 재시작 또는 재실행 후에도 상태가 유지되는가? |  |  |
| 유지보수성 | 다음 세션이 저장소 문서만으로 이해할 수 있는가? |  |  |
| 보안 | 비밀, 권한, 외부 행동 규칙을 지켰는가? |  |  |

## 판정

- Accept: 모든 필수 항목이 검증됐고 주요 위험이 없습니다.
- Revise: 핵심 방향은 맞지만 누락된 검증이나 문서 갭이 있습니다.
- Block: 완료 증거가 없거나, 범위를 벗어났거나, 되돌리기 어려운 위험이 있습니다.
